# Запуск модели для общения

Эта папка содержит код для запуска модели в терминале для интерактивного общения.

## Использование

### Базовый запуск (только базовая модель):

```bash
python chat.py --base_model "microsoft/DialoGPT-small"
```

### Запуск с дообученной LoRA моделью:

```bash
python chat.py \
    --base_model "microsoft/DialoGPT-small" \
    --lora_model "../fine_tuning/lora_model"
```

### С дополнительными параметрами:

```bash
python chat.py \
    --base_model "microsoft/DialoGPT-small" \
    --lora_model "../fine_tuning/lora_model" \
    --system_prompt "Ты полезный ассистент." \
    --max_length 512 \
    --temperature 0.7
```

## Параметры

- `--base_model`: Имя базовой модели с HuggingFace (обязательно)
- `--lora_model`: Путь к дообученной LoRA модели (опционально)
- `--system_prompt`: Системный промпт для модели (опционально)
- `--max_length`: Максимальная длина генерируемого ответа (по умолчанию: 512)
- `--temperature`: Температура генерации, контролирует случайность (по умолчанию: 0.7)
  - Более низкие значения (0.1-0.5) делают ответы более детерминированными
  - Более высокие значения (0.7-1.0) делают ответы более креативными

## Команды в чате

- `quit`, `exit` или `q` - выход из чата
- `clear` - очистка истории разговора

## Пример использования

```
Вы: Привет! Как дела?
Модель: Привет! У меня все отлично, спасибо! Как дела у тебя?

Вы: Расскажи анекдот
Модель: [модель генерирует анекдот]

Вы: quit
До свидания!
```

## Примечания

- Модель автоматически использует GPU если доступен
- История разговора ограничена последними 10 сообщениями для экономии памяти
- Если указан путь к LoRA модели, веса будут объединены с базовой моделью
